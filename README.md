# ink-foreman

An [ink](https://github.com/vadimdemedes/ink) terminal application for managing [foreman](https://github.com/ddollar/foreman) logs.

<img src="./example.png" alt="Example" width="100%">

## Install

```bash
$ git clone https://github.com/ericmatte/ink-foreman
$ npm install
$ npm link
```

## CLI

```
$ ink-foreman --help

  Usage
    $ ink-foreman
```

_Note: This app may flicker on some terminals (ie.: iterm2)_
